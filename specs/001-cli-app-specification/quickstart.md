# Quickstart Guide: Python CLI Application

## Getting Started

### Prerequisites
- Python 3.10 or higher
- Terminal/command prompt access

### Running the Application
1. Open your terminal or command prompt
2. Navigate to the directory containing `main.py`
3. Run the application with the command:
   ```bash
   python main.py
   ```

### Basic Usage
- Once the application starts, you'll see a command prompt
- Enter commands as prompted by the application
- The application will process your commands and display results
- To exit, use the exit command or press Ctrl+C

### Example Commands
- `help` - Display available commands
- `exit` - Quit the application
- Other commands will be specific to the application's functionality

### Troubleshooting
- If you get a "Python not found" error, ensure Python 3.10+ is installed and in your PATH
- If the application crashes, ensure you're using Python 3.10 or higher
- For invalid command errors, check the help command for available options
